NASA 3D Asteroid Viewer

Background

  • The mission was to create a viewer that reads data from NASAâ€™s database to render a real 3D model of an asteroid. The user should be able to move the asteroid around to view its surface and click on specific points on the asteroid to receive the data of these points.
  • Our team successfully created a working viewer that renders a real model of several existing asteroids where the user is able to move them around with their cursor and click on specific points for data retrieval.
  • Various tools were used in order to create the viewer, including HTML, Javascript, CSS, etc. For example, a Javascript library called Three.js was used to create a 3D illusion of stars surrounding the asteroids with the use of 2D images.
  • This project was liked by several workers at NASA during a public presentation that NASA adapted the project to release for public use in the future.
